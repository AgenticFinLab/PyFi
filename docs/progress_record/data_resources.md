
# Data resources for Financial Trend Charts or Figures

## Book Retrieval Platfrom
- Digital Book Platforms
  - [Google Books](https://books.google.com/)  
    Billed as the world's largest book collection

  - [Zlibrary](https://z-library.cc/)  
    Billed as the world's largest e-book library
  
  - [O-lib](https://github.com/shiyi-0x7f/o-lib) (Yuxuan found)

  - [Internet Archive Texts](https://archive.org/details/texts)**  
Offers over 20 million downloadable publications (40+ million resources total)  
    - Related: [OpenLibrary](https://openlibrary.org/) - Aims to include "every book ever published"

- Library Catalogs & Alliances
  - [WorldCat](https://search.worldcat.org/)  
    World's largest library catalog

  - [National Library Reference and Consultation Alliance](http://www.ucdrs.cn/)    
    Non-profit literature service led by Guangdong Provincial Sun Yat-sen Library  
    - Official: [Union Rules](http://www.ucdrs.cn/)  
    - Access: [SuperLib](http://www.ucdrs.superlib.net/) | [ZSLIB](http://search.zslib.com.cn/)  
    - Portal: [ZSLIB Main](https://www.zslib.com.cn/)

  
- Specialized Libraries
  - **[IMF eLibrary](https://www.elibrary.imf.org/)** 
  International Monetary Fund online library

  - **[World Bank E-Library](http://www.worldbank.org/elibrary)**
